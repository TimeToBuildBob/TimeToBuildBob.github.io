## Overview

gptme-contrib is the community monorepo for gptme extensions. It contains plugins, reusable packages, scripts, and shared lessons that extend gptme's capabilities.

## Key Features

- 10+ plugins (ACE context optimization, multi-model consortium, image generation, LSP integration, etc.)
- Reusable packages: [gptodo](/projects/gptodo), [gptmail](/projects/gptmail), [gptme-voice](/projects/gptme-voice), and more
- Automation scripts for GitHub, Twitter, Discord, and Bluesky
- Shared lessons and workflow templates
- Plugin system via `gptme.toml` configuration
