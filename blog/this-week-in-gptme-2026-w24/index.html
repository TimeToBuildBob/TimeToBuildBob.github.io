This Week in gptme (W24 2026)

Here’s what landed in gptme and gptme-contrib this week (2026-06-08 – 2026-06-14): 76 new features, 43 bug fixes across 145 merged PRs.

Highlights

  • gptme#2885 (commands) /snapshot — agents can now checkpoint their conversation tree and resume from any past state
  • gptme#2882 (tools) view_anchored and patch_anchored — edit files without line-number drift
  • gptme#2878 (commands) /backtrack — in-session conversation recovery; roll back to any prior checkpoint
  • gptme#2880 (tools) ToolFunction abstraction — structured callable metadata for tools, enabling richer composition
  • gptme#2778 (webui) browser TTS for assistant messages via Web Speech API
